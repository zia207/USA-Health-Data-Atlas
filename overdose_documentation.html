CONUS County-Level Drug Overdose & Opioid Mortality — Dataset Documentation

County-level drug-overdose and opioid-related mortality for the contiguous United States (48 states + DC). The USA Overdose Mortality explorer (usa_overdose_mortality.html) maps three live CDC/NCHS source layers; a separate synthetic long-format panel (conus_overdose_timeseries.parquet) remains available for offline pipeline development and SQL prototyping.

Synthetic parquet panel (offline development)

conus_overdose_timeseries.parquet is a tidy long-format panel (1999–2023) where mortality values are synthetic (SYNTHETIC_DEMO), generated by overdose_mortality_pipeline.py so analysis code can be developed against a realistic schema. Use the live CDC pull in the explorer or build_timeseries(use_synthetic=False) for authoritative figures.

The current model-based atlas layer uses posterior median model-based death rates from rpvx-m2md (INLA hierarchical Bayesian models with spatial and temporal random effects). An older pipeline connector (pbkm-d27e) used binned age-adjusted rate bands; that file is retained for compatibility but is not what the explorer maps.

Opioid county allocation (explorer)

County-level opioid-specific counts are not published on data.cdc.gov and CDC WONDER’s public API does not support automated county-level queries. The opioid tab therefore:

  1. Pulls state 12-month-ending opioid death totals from xkb8-kh2a (indicator: Opioids (T40.0-T40.4,T40.6)), using December values per year.
  2. Pulls county provisional all-drug overdose counts from gb4e-yj24 (unsuppressed cells only).
  3. Allocates each state’s opioid total to counties in proportion to each county’s share of the state’s provisional all-drug overdose deaths:
county_opioid_est = state_opioid × (county_provisional_all / sum(county_provisional_all in state))

This preserves state opioid totals and yields plausible relative county patterns, but is an estimate — not an observed county opioid count.

Suppression semantics

Provisional tab (gb4e-yj24): NCHS suppresses counts of 1–9. Suppressed county-years are flagged (suppressed = true) and excluded from the choropleth. Roughly half of county-years may be suppressed; do not sum suppressed cells for totals.

Model-based tab (rpvx-m2md): Rates are model-based estimates with credible intervals; no small-count suppression, but values may differ from WONDER observed counts.

Opioid tab: Only counties with unsuppressed provisional all-drug counts receive an allocated opioid estimate.

Synthetic generation (parquet only)

When use_synthetic=True, expected deaths for a county × year are drawn as a Poisson variable around an intensity curve calibrated to the real US epidemic:

  • Rising baseline across 1999–2023.
  • Fentanyl-era acceleration from 2013 onward.
  • Pandemic bump from 2020 onward.
  • Per-county heterogeneity via a log-normal place effect.
  • Rising opioid share of overdose deaths over time.
  • Registry suppression (1–9) applied to the provisional-count layer.
  • Reproducibility: fixed seed (207).

The generator targets realistic shape, not truth.

Usage

Atlas / cached CDC parquets

import pandas as pd

model = pd.read_parquet("conus_overdose_model_based_cdc.parquet")
prov  = pd.read_parquet("conus_overdose_provisional_cdc.parquet")
opi   = pd.read_parquet("conus_overdose_opioid_cdc.parquet")

# Tompkins County, NY — model-based trajectory
tompkins = model.query("fips == '36109'").sort_values("year")

Synthetic long panel

df = pd.read_parquet("conus_overdose_timeseries.parquet")

wide = (df.query("category == 'drug_overdose_all'")
          .pivot_table(index=["fips", "year"], columns="source",
                       values="deaths"))

Caveats and limitations

  • Source seam ~2019–2021. Model-based estimates end in 2021; provisional counts begin in 2020. Do not treat these as one continuous series without reconciliation.
  • Opioid county tab is estimated, not WONDER-observed. Use for relative geography, not exact county opioid totals.
  • Don’t sum suppressed provisional cells — they undercount.
  • Model-based rates are estimates from Bayesian models; credible intervals are provided on the model-based tab.
  • Population denominators in the explorer use conus_counties.csv (2018 baseline) for provisional and opioid crude rates; model-based tab uses population from CDC.
  • CONUS only. Alaska, Hawaii, and territories are excluded by design.
  • Synthetic parquet remains for offline development; the explorer uses live CDC data.
